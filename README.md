# 项目概述 python987

本项目是一款基于Python的博客管理系统，采用Django框架进行开发，使用HTML、CSS、JS进行前端设计，以及MySQL作为数据库支持。

## 技术栈

### 后端技术
- Python
- Django

### 前端技术
- HTML
- CSS
- JavaScript

### 数据库
- MySQL

## 运行环境

- 开发工具：PyCharm
- 数据库：MySQL 5.7
- 数据库管理工具：Navicat 10 或更高版本
- 环境配置软件：Python 3.8

## 功能模块
- 用户管理
- 博客发布与编辑
- 评论管理
- 分类与标签管理

## 系统角色
- 系统管理员
- 博主
- 访问者

## 部署步骤
1. 安装Python 3.8及其以上版本
2. 安装MySQL 5.7并配置数据库
3. 使用Navicat管理数据库
4. 克隆或下载项目代码到本地
5. 安装依赖项 `pip install -r requirements.txt`
6. 运行 `python manage.py runserver` 启动开发服务器

## 核心亮点
- 基于Python和Django，易于扩展和维护
- 前端使用HTML、CSS、JS，响应式设计，适用于多种设备
- 采用MySQL数据库，数据存储稳定可靠
